What is this?

Welcome to API Sea by Sergie, a tool desgined to explore existing API's and see how they can be added to a project.

Here you will be able to find API's from a variety of sources that may inspire new ideas on how to use them on a personal project. The version on the code hosted on GitHub does not have any API keys that may be needed to use them. You need to visit the official documentation and generate a key for yourself.

This is an ongoing project being constantly updated. Come back regularly for new exciting features and the latest knowledge on web development!

How to use the site

Exploring a project

Simply select any of the cards on the Explore Menu and see how the API works. A brief description, link to the official documentation, an example of application and additional comments will be displayed. The "complexity of use" provides a general view of how difficult it would be for a beginner to use the API. When the difficulty is "variable", it means the API has several ways to be used, with varying difficulties.
